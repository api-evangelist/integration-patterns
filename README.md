# Integration Patterns (integration-patterns)
Integration Patterns are design patterns and best practices for integrating different software systems and applications, including messaging, data transformation, and service orchestration approaches. Enterprise Integration Patterns (EIP) provide a comprehensive catalog of patterns for building robust, scalable integration solutions.
